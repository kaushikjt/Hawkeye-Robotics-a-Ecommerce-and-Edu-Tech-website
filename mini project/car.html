Bluetooth Controlled Car

Description:

This project involves building a remote-controlled car that can be wirelessly operated via Bluetooth communication from a mobile device. Users can control the car's movement and possibly other functions using a smartphone or tablet.

Components Required:

  • Chassis: Provides the structure and wheels for the car.
  • DC Motors: Typically two motors for driving the car forward, backward, and turning.
  • Motor Driver: Module (e.g., L298N) to control the speed and direction of the motors.
  • Bluetooth Module: Such as HC-05 or HC-06 for wireless communication with the mobile device.
  • Microcontroller: Like Arduino or ESP32 to interface with the Bluetooth module and control the motor driver.
  • Power Supply: Battery pack to power the motors and electronics.
 code:
// Define motor control pins
const int motorLeftPin1 = 2;
const int motorLeftPin2 = 3;
const int motorRightPin1 = 4;
const int motorRightPin2 = 5;

// Define Bluetooth communication pins
const int bluetoothTx = 1; // Arduino TX to Bluetooth RX
const int bluetoothRx = 0; // Arduino RX to Bluetooth TX

// Variables to store incoming commands from Bluetooth
char command;

void setup() {
  // Initialize motor control pins as outputs
  pinMode(motorLeftPin1, OUTPUT);
  pinMode(motorLeftPin2, OUTPUT);
  pinMode(motorRightPin1, OUTPUT);
  pinMode(motorRightPin2, OUTPUT);
  
  // Initialize Serial communication for Bluetooth
  Serial.begin(9600);
}

void loop() {
  // Check if data is available to read from Bluetooth
  if (Serial.available() > 0) {
    command = Serial.read(); // Read the incoming command
    
    // Perform actions based on the command received
    if (command == 'F') {
      // Move forward
      digitalWrite(motorLeftPin1, HIGH);
      digitalWrite(motorLeftPin2, LOW);
      digitalWrite(motorRightPin1, HIGH);
      digitalWrite(motorRightPin2, LOW);
    } else if (command == 'B') {
      // Move backward
      digitalWrite(motorLeftPin1, LOW);
      digitalWrite(motorLeftPin2, HIGH);
      digitalWrite(motorRightPin1, LOW);
      digitalWrite(motorRightPin2, HIGH);
    } else if (command == 'L') {
      // Turn left
      digitalWrite(motorLeftPin1, LOW);
      digitalWrite(motorLeftPin2, HIGH);
      digitalWrite(motorRightPin1, HIGH);
      digitalWrite(motorRightPin2, LOW);
    } else if (command == 'R') {
      // Turn right
      digitalWrite(motorLeftPin1, HIGH);
      digitalWrite(motorLeftPin2, LOW);
      digitalWrite(motorRightPin1, LOW);
      digitalWrite(motorRightPin2, HIGH);
    } else if (command == 'S') {
      // Stop
      digitalWrite(motorLeftPin1, LOW);
      digitalWrite(motorLeftPin2, LOW);
      digitalWrite(motorRightPin1, LOW);
      digitalWrite(motorRightPin2, LOW);
    }
  }
}

The Bluetooth module is connected to the microcontroller (e.g., Arduino) using serial communication (TX and RX pins). The microcontroller controls the motor driver module, which in turn drives the DC motors based on commands received via Bluetooth from the mobile device.
Power is supplied to the motors through the motor driver module and to the microcontroller and Bluetooth module through the power supply.

Applications:

  • Education and Learning: Provides hands-on experience with electronics, microcontrollers, and Bluetooth communication.
  • Fun and Hobby: Enjoyable for hobbyists interested in building and controlling robotic vehicles.
  • Prototyping: Can serve as a prototype for more advanced autonomous or remotely operated vehicles.
