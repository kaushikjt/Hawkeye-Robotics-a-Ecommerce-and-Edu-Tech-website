Line Follower Robot

Description:

This project involves building a robot that can autonomously follow a line marked on the ground using infrared sensors. The robot detects the contrast between the line and the surface and adjusts its path to stay centered on the line.

Components Required:

  • Chassis: Provides the structure and wheels for the car.
  • DC Motors: Typically two motors for driving the car forward, backward, and turning.
  • Motor Driver: Module (e.g., L298N) to control the speed and direction of the motors.
  • Infrared Sensors: Typically 2 or more sensors to detect the line.
  • Microcontroller: Like Arduino or ESP32 to interface with the Bluetooth module and control the motor driver.
  • Power Supply: Battery pack to power the motors and electronics.
 code:
 // Define pin numbers for sensors
 const int sensorLeftPin = 2;
 const int sensorRightPin = 3;
 
 // Define pin numbers for motor driver
 const int motorLeftPin1 = 4;
 const int motorLeftPin2 = 5;
 const int motorRightPin1 = 6;
 const int motorRightPin2 = 7;
 
 void setup() {
   // Initialize sensor pins as inputs
   pinMode(sensorLeftPin, INPUT);
   pinMode(sensorRightPin, INPUT);
   
   // Initialize motor driver pins as outputs
   pinMode(motorLeftPin1, OUTPUT);
   pinMode(motorLeftPin2, OUTPUT);
   pinMode(motorRightPin1, OUTPUT);
   pinMode(motorRightPin2, OUTPUT);
 }
 
 void loop() {
   int sensorLeftValue = digitalRead(sensorLeftPin);
   int sensorRightValue = digitalRead(sensorRightPin);
   
   // Adjust motor directions based on sensor readings
   if (sensorLeftValue == HIGH && sensorRightValue == LOW) {
     // Turn left
     digitalWrite(motorLeftPin1, LOW);
     digitalWrite(motorLeftPin2, HIGH);
     digitalWrite(motorRightPin1, HIGH);
     digitalWrite(motorRightPin2, LOW);
   } else if (sensorLeftValue == LOW && sensorRightValue == HIGH) {
     // Turn right
     digitalWrite(motorLeftPin1, HIGH);
     digitalWrite(motorLeftPin2, LOW);
     digitalWrite(motorRightPin1, LOW);
     digitalWrite(motorRightPin2, HIGH);
   } else if (sensorLeftValue == LOW && sensorRightValue == LOW) {
     // Move forward
     digitalWrite(motorLeftPin1, HIGH);
     digitalWrite(motorLeftPin2, LOW);
     digitalWrite(motorRightPin1, HIGH);
     digitalWrite(motorRightPin2, LOW);
   } else {
     // Stop (optional fallback)
     digitalWrite(motorLeftPin1, LOW);
     digitalWrite(motorLeftPin2, LOW);
     digitalWrite(motorRightPin1, LOW);
     digitalWrite(motorRightPin2, LOW);
   }
 }

In the circuit, infrared sensors are positioned close to the ground to detect the line. They are connected to the analog or digital input pins of the microcontroller (e.g., Arduino). The microcontroller processes sensor inputs and sends commands to the motor driver module to adjust motor speeds or directions based on the line-following algorithm.
Power is supplied to the motors through the motor driver module and to the microcontroller and Bluetooth module through the power supply.

Applications:

  • Education: Used in STEM education to teach concepts of sensors, microcontrollers, and autonomous robotics.
  • Industrial: Deployed in factories for material handling and automated guided vehicles (AGVs).
  • Hobbyist Projects: Popular among robotics enthusiasts for experimentation and learning.
